# ST0C

ST0C é o projeto principal deste workspace.

## Visão geral

ST0C é desenvolvido com foco em **arquitetura limpa, simples e escalável**. Todo o
desenvolvimento real acontece dentro desta pasta (`ST0C/`).

## Estrutura do workspace

```
C/
├── ST0C/         # Projeto principal (este repositório)
├── gstack-main/  # Ferramenta / skill pack auxiliar (NÃO é o projeto principal)
└── c.com/        # Outros arquivos do workspace
```

## Ferramentas auxiliares

- **gstack-main** — usado apenas como referência / ferramenta / skill pack. Não é o
  projeto principal e seu código não deve ser misturado dentro do ST0C sem necessidade.
- **ruflo** — agente / MCP auxiliar usado por trás para apoiar o desenvolvimento.

## Princípios

1. ST0C é sempre o projeto principal.
2. gstack-main e ruflo são apenas ferramentas de apoio.
3. Antes de alterar código, o plano deve ser explicado.
4. A arquitetura deve permanecer limpa, simples e escalável.

## Como começar

Este repositório está em fase inicial de configuração. As próximas etapas de
desenvolvimento serão adicionadas conforme o projeto evoluir.
